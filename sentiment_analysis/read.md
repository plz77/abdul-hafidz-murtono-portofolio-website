# 🧠 AI Sentiment Analysis Hub (Bahasa Indonesia)

Aplikasi klasifikasi teks mutakhir (*state-of-the-art*) untuk menganalisis dan membedakan polaritas emosi (Sentimen Positif vs Negatif) khusus teks tertulis Bahasa Indonesia.

## ⚡ Fitur Utama
- **100% Client-Side Machine Learning**: Menjalankan model deep learning tingkat lanjut langsung di dalam web browser pengguna menggunakan runtime WebAssembly (`Transformers.js`). Menjamin privasi data pengguna secara absolut.
- **Indonesian Semantic Core**: Memanfaatkan model BERT teroptimasi `Xenova/indonesian-sentiment-analysis` yang telah di-fine-tune khusus untuk mendeteksi linguistik dan bahasa slang Indonesia.
- **Confidence Tracker Bar**: Visualisasi persentase tingkat keyakinan (*confidence score*) model kecerdasan buatan terhadap hasil klasifikasi teks yang dimasukkan.
- **Multilingual UI & Dual Theme**: Desain premium bertema Biru Profesional yang dilengkapi tombol alih bahasa (ID/EN) serta toggle Dark/Light Mode.

## 💻 Spesifikasi Teknis
- **AI Core:** Transformers.js v2.6.0 (Hugging Face Javascript Porting).
- **Engine Framework:** ONNX Runtime Web.
- **UI Framework:** Tailwind CSS dengan konfigurasi mode gelap berbasis class.
