
# 🖼️ Folder Aset Gambar

Folder ini merupakan direktori pusat untuk menyimpan semua aset media visual yang digunakan dalam ekosistem website portofolio profesional ini. Penggunaan folder ini bertujuan untuk menjaga struktur repositori tetap rapi dan terorganisir.

## 📂 Struktur dan Isi Folder

Folder ini mencakup berbagai jenis file gambar, termasuk:

- **Aset Profil**: Foto profil resmi dan variasi gambar representasi diri (misalnya: `foto-diri-1.jpg`, `A. Hafidz.jpg`).
- **Ikon & Logo**: Grafik logo untuk representasi lembaga atau teknologi (misalnya: `logo.webp`, `Kampus Merdeka.jpg`).
- **Media Proyek**: Tangkapan layar (*screenshots*) atau visual pendukung untuk mikro-aplikasi portofolio, seperti demo fitur analisis sentimen atau kalkulator KPR (misalnya: `sentimen-demo.jpg`).
- **Grafis Blog**: Gambar *thumbnail* atau banner untuk bagian publikasi/blog (misalnya: `blog-1.jpg`, `blog-2.jpg`).

## 🛠️ Panduan Penggunaan dan Kontribusi

Untuk menjaga integrasi yang mulus dengan file kode utama (seperti `index - Copy.html`), harap perhatikan pedoman berikut:

1. **Pemanggilan File**: Gunakan jalur relatif yang benar saat memanggil gambar dalam kode HTML, misalnya `<img src="./images/nama-file.jpg" alt="...">`.
2. **Penamaan File**: Gunakan nama file yang deskriptif dan hindari penggunaan spasi berlebihan (disarankan menggunakan *kebab-case* seperti `nama-file.jpg`).
3. **Format & Optimasi**: Sangat disarankan untuk mengunggah gambar yang sudah dioptimalkan (dikompres) dalam format modern seperti `.webp` atau `.jpg` untuk memastikan kecepatan pemuatan halaman yang optimal.
