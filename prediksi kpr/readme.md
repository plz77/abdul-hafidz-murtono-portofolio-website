# 📊 Mortgage Engine Calculator (Prediksi KPR)

Aplikasi kalkulator finansial presisi yang digunakan untuk memprediksi, mengalkulasi, dan menganalisis simulasi angsuran Kredit Pemilikan Rumah (KPR) secara terstruktur.

## ⚡ Fitur Utama
- **Precision Amortization Calculations**: Algoritma matematika finansial yang menghitung kalkulasi rasio cicilan bulanan secara tepat berdasarkan variabel tenor dan suku bunga.
- **Custom Asset Evaluations**: Fleksibilitas penyesuaian nilai uang muka (*down-payment*) dan kalkulasi harga properti secara dinamis.
- **Clean Split-Page Architecture**: Pemisahan modul input formula (`index.html`) dan halaman pelaporan output ringkasan (`kpr_output.html`) untuk menjaga kebersihan arsitektur kode.

## 💻 Spesifikasi Teknis
- **UI Styling:** Tailwind CSS Modern Light-Vibrant theme.
- **Logic Controller:** JavaScript DOM Manipulation dengan manajemen penyimpanan data sementara (*temporary local storage pass*).
